Huh, 2015

今年不想再定所谓的新年计划了，每次悄悄的写在笔记里一个没实现然后默默删掉。
Shame on you..
今年能多记录下生活的点滴和感悟就可以了。
元旦前几天，大半夜的睡不着觉，凌晨四点端个相机出门逛西湖，冻成狗
第一次夜游杭州。在这个熟悉又陌生的城市里混了七年，却没有太多归属感。
每次回家也是一样，熟悉的地方，感觉却很陌生。
这就是自然法则，是时候自己出去闯，寻求自己的事业和家庭。
不要害怕孤单，要耐得住寂寞。
说说几个愿望吧。
1、offer offer offer！！！来个CMU，UCLA，UCSD啥的收了我吧！！！
2、多看几本书，顺便练练字啥的
3、全家人身体健康吧，我的话，有空常锻炼。
4、旅行旅行！Guitar捡起来~
5、最后，顺利毕业！
BTW，happy new year！

Live curious

最近晚上总睡得太晚，想些有的没的。
临近毕业，似乎看不到清晰的未来。
想起国家地理曾有段宣传片“Live Curious”。
即使迷茫，也不能失去对美好世界的好奇心。
Move on, enjoy your life.

If you are alive, you breath.

If you breath, you talk.

If you talk, you ask.

If you ask, you think.

If you think, you search.

If you search, you experience.

If you experience, you learn.

If you learn, you grow.

If you grow, you wish.

If you wish, you find.

If you find, you doubt.

If you doubt, you question.

If you question, you understand.

If you understand, you know.

If you know, you want to know more.

If you want to know more, you are alive.

Live curious.

有朋自远方来

昨天下午教四改PS，突然接到澄哥电话，晚上聚餐，果断帮着电话联络大家，搞起！
晚饭自然是吃的很High，8个人黄酒搞了7瓶。作为一个从不喝黄酒的人，我也喝了大半瓶。晚上KTV太high，又是30瓶冰锐，还有各自鬼哭狼嚎杀猪般献歌，还得群姐出来镇场子。。
真的很开心。
或许是学校一个人待太久，太希望有人陪着玩闹。
或许是心里压得事情太多，需要发泄。
但其实，是同窗这份友谊太难得。
十年二十年之后，大家聚在一起，若还能如此般情真意切，那便是福分。
得一知己难。有朋如此需珍惜。
By Liz, 12/6/2014.
